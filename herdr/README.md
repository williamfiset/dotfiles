# herdr

Config for [herdr](https://herdr.dev) (terminal multiplexer / agent manager).

## Install

```
mkdir -p ~/.config/herdr
cp config.toml ~/.config/herdr/config.toml
```

Reload with `prefix+shift+r` in a running session.

Notable settings: dracula theme, cyan accent, pane borders/gaps on,
`ctrl+arrows` pane navigation, terminal toast notifications.
