# dotfiles

My personal computer configuration files — the settings, shell config, and setup
scripts I carry between machines to get a new Mac feeling like home.

## Contents

| Folder | What's inside |
| --- | --- |
| [`computer-setup/`](computer-setup/) | Steps and software for setting up a new Mac |
| [`fish-shell/`](fish-shell/) | Fish shell config, prompt, and git prompt sync |
| [`git-hooks/`](git-hooks/) | Global git hooks that keep shell prompts in sync |
| [`obsidian/`](obsidian/) | Obsidian theme and community plugin setup |
| [`terminal-multiplexers/`](terminal-multiplexers/) | tmux and herdr configuration |
| [`terminals/`](terminals/) | Ghostty and iTerm2 configuration |
| [`vim/`](vim/) | Vim configuration (`vimrc`) |
| [`archive/`](archive/) | Previous tools and configs no longer in my primary workflow |

Each folder has its own README with setup and usage details.
